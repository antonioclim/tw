Tema: Implementarea unui Ping → Pong minimal

În cadrul acestui exercițiu, veți învăța cum să comunicați cu un server printr-un request de tip GET. Avem două moduri de testare:

1. Interfață cu buton

Dă click pe butonul de mai jos. Browserul va trimite un request GET către /ping, iar serverul va răspunde cu pong.

2. Test din Consola Developer

Deschide Developer Tools (tasta F12 sau Ctrl+Shift+I), mergi în tab-ul Console și rulează următorul cod: 


fetch("/ping")
  .then(r => r.text())
  .then(console.log);

Ar trebui să vezi în consolă: pong.

Instrucțiuni pentru server

Pentru ca acest exemplu să funcționeze, ai nevoie de un server Node.js simplu cu Express:


// Salvează ca server.js
import express from "express";
const app = express();
const port = 3000;

app.get("/ping", (req, res) => {
  res.send("pong");
});

app.listen(port, () => {
  console.log(`Server pornit la http://localhost:${port}`);
});

Rulează serverul cu comenzile:
npm install express
node server.js
