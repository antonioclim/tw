\# S11 – React: arhitectură, laborator și kit-uri „Movie App”



Acest repository conține materialele complete pentru \*\*Seminarul 11\*\* din cadrul

seminarului 11 de \*\*Tehnologii Web\*\*. Este centrat pe introducerea și utilizarea

\*\*React\*\* într-un context modern \_single-page application\_ (SPA) și \_full‑stack\_.



Pachetul este structurat astfel încât să acopere:



\- \*\*teorie\*\* (arhitectura și fundamentele React);

\- \*\*laborator ghidat\*\* (starterkit React cu pași 1–5 și teme);

\- \*\*anexă extinsă\*\* (4 kit-uri „Movie App” + resurse audio/video și HTML).
